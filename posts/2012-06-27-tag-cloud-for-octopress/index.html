Tag cloud for Octopress

Mon, Jan 1, 0001

Octopress的預設畫面沒有將分類標籤獨立出來，讓搜尋文章類別很不方便，因此標籤雲這東西就相對的重要，網路上找了一下相關的plugin，以下說明一下安裝與設定過程。

Installation steps

  1. Download plugin
  2. Move file
  3. Edit config
  4. Compile octopress

Download plugin

  1. 下載Plugin相關檔案 GitHub - tokkonopapa/octopress-tagcloud
  2. Plugin相關檔案

Move file

將檔案移動到Octopress對應的位置

Edit config

  1. 編輯_config.yml default_asides 是設定Blog側邊欄所要放置的模板，我選擇了tag_cloud.html去作類別的顯示；
    使用的模板位置 custom/asides/tag_cloud.html
